
Package slic3r 3.0-dev and move it to our app

1. Install PAR::Packer
  cpan App::cpanminus
  cpanm PAR::Packer

2. In slic3r folder
  A. pp -e -p -x slic3r.pl --xargs cube.stl -o slic3r.par
  B. mv slic3r.par slic3r.zip
  C. Extract
  D. cp slic3r.pl to the new folder. This folder as all the required files for slic3r
  E. The static libraries for command line slic3r are only Boost.
  F. find ./ -name '*.bundle'
     otool -L somefile.bundle
